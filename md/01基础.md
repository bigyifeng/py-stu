### 数据类型

| 字段  | 描述                              |
| ----- | --------------------------------- |
| str   | 字符串                            |
| int   | 整数                              |
| float | 小数                              |
| tuple | 元组（不可改的 list）             |
| list  | 列表                              |
| set   | 集合（值唯一的 list，会自动去重） |
| dict  | 字典                              |
